Je suis un grand fan de Clash of Clans depuis plusieurs années.</br>
J’adore le jeu et je possède de nombreux comptes, ce qui me permet de tester différentes stratégies et styles de jeu.

En parallèle, j’ai une vraie passion pour le développement et le codage. J’aime créer des projets qui combinent logique, automatisation et créativité. Pour allier ces deux centres d’intérêt, j’ai décidé de me lancer dans la création d’un bot pour Clash of Clans, un projet qui me permet à la fois de programmer et de mieux comprendre la mécanique du jeu.
</br></br></br>

# OBJECTIFS

L'objectif à long terme serait de réussir à créer toutes les features nécessaires pour jouer au jeu complet presque automatiquement.

Je vais commencer par les features les plus simples comme le farm de ressources dans la maison des ouvriers (MDO) en finissant par la réalisation des jeux de clans (JDC).

Voici la liste des features à ce jour : </br>
✅ Farm MDO </br>
❌ Farm Village principal </br>
❌ Amélioration murs automatique </br>
❌ Attaques de raids </br>
❌ Auto demande de troupes </br>
❌ Auto donation de troupes </br>
❌ Jeux de clans 


</br></br></br>

# UTILISATION

Pour utiliser le bot vous pouvez simplement :
- Cloner le repo
- Installer les dépendances de __"requirements.txt"__
- Lancer le script __"KisaPulBot_COC.py"__
